# DunDefCalc

DunDefCalc is a calculating tool for Dungeon Defenders. It's written in javascript and runs the calculations locally.<br>
<br>
You can find a hosted version [here](https://guanab.github.io/dundefcalc/) or you can download the index.html and itemdata.json files from this repo and host them yourself. An easy way to do this is for example Python http.server module.<br>
